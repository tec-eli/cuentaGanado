# Cattle Counter

Minimal, frictionless Android app for real-time cattle counting in the field.

## Purpose
Count cattle (male/female) quickly and reliably, with zero friction and no distractions.

## Usage
1. Open the app
2. Tap buttons to count
3. Reset with long press or double tap (with confirmation)
4. App state is saved automatically

## Development
- Kotlin + Android Studio
